# 🎓 Student Management System

A console-based Student Management System developed using **Java**, **JDBC**, and **MySQL**. This application allows a college administrator to manage student records efficiently through a menu-driven interface.

## 🚀 Features

- ➕ Add Student
- 📋 View All Students
- 🔍 Search Student by ID
- ✏️ Update Student Details
- 🗑️ Delete Student
- 📊 Manage Student Records using MySQL Database
- ✅ Menu-driven Console Interface

## 🛠️ Technologies Used

- Java
- JDBC
- MySQL
- VS Code
- Git & GitHub

## 🗄️ Database

Database Name:

```
collegeDB
```

Table:

```
student
```

Fields:

- Student ID
- Student Name
- Age
- Gender
- Course
- Email
- Phone Number
- Marks

## ▶️ How to Run

1. Clone the repository.

```
git clone https://github.com/your-username/StudentManagementSystem-JDBC.git
```

2. Open the project in VS Code.

3. Create the MySQL database and `student` table.

4. Update the database credentials in `DBConnection.java`.

5. Run `Main.java`.
